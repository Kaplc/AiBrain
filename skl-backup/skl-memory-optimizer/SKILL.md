# 记忆优化技能 (skl-memory-optimizer)

将用户想保存的信息，经过分类、去重、语义扩展后，以标准格式存入 qdrant-memory，让日后的搜索更准确、更全面。

---

## 核心思路

qdrant-memory 靠**语义向量**来匹配，存入的文字质量直接决定搜索精度：
- **太简短**（"喜欢苹果"）→ 语义信息少，搜"你吃什么水果"可能找不到
- **有分类前缀**（"[饮食偏好] 喜欢苹果"）→ 同类信息聚合，搜索更准
- **适度扩展同义词**（"苹果、果类、水果"）→ 不同表达都能命中
- **去重**（已存过的不重复存）→ 避免噪声干扰搜索结果

---

## 记忆分类体系

存储时为每条记忆打上分类标签，便于语义聚合：

| 分类标签 | 适用内容 |
|---------|---------|
| `[个人信息]` | 姓名、年龄、职业、城市等基础信息 |
| `[饮食偏好]` | 喜欢/不喜欢的食物、饮料、口味 |
| `[运动习惯]` | 喜欢的运动、锻炼频率、健身方式 |
| `[兴趣爱好]` | 游戏、音乐、阅读、影视等休闲偏好 |
| `[工作学习]` | 职业技能、学习目标、工作相关 |
| `[生活习惯]` | 作息、日常规律、生活方式 |
| `[人际关系]` | 家人、朋友、同事相关信息 |
| `[目标计划]` | 短期/长期目标、待办事项、计划 |
| `[其他]` | 不属于以上类别的信息 |

## 操作一：保存新记忆

当用户说"记住 XX"或"帮我记一下 XX"时：

### 步骤

1. **理解用户意图**
   - 提取核心事实（去掉"帮我记住"、"记一下"等指令词）
   - 判断分类标签

2. **检查是否重复**
   - 调用 `mcp__qdrant-memory__search`，用核心内容搜索
   - 如果找到 score > 0.85 的相似记忆，说明已有相关记录
   - 提示用户："已有相似记忆：XXX，是否更新？"

3. **构造优化后的记忆文本**

   格式模板：
   ```
   [{分类标签}] {核心事实的清晰陈述}
   扩展: {3~5个紧密相关的同义词或上位词}
   ```

   示例：
   ```
   用户输入: "记住我喜欢走路"
   
   存储内容:
   [运动习惯] 用户喜欢走路
   扩展: 步行、散步、徒步、有氧运动
   ```

   ```
   用户输入: "记住我叫志远"
   
   存储内容:
   [个人信息] 用户的名字叫志远
   扩展: 姓名、称呼、名字
   ```

4. **调用 `mcp__qdrant-memory__store`** 存入
5. **反馈给用户**：
   - 新存：✅ 已记住：[运动习惯] 用户喜欢走路
   - 更新：✅ 已更新记忆（原：XXX → 新：XXX）
   - 跳过：ℹ️ 已有相同记忆，无需重复存储

---

## 操作二：批量整理已有记忆

当用户说"整理一下我的记忆"或"清理重复记忆"时：

### 步骤

1. **获取所有记忆**
   - 调用 `mcp__qdrant-memory__list_all` 获取全部记忆

2. **分析问题**
   - **重复项**：内容高度相似（如"用户喜欢跑步"存了3条）
   - **格式混乱**：没有分类标签的旧记忆
   - **信息冲突**：同一主题有矛盾的记录

3. **制定整理方案并告知用户**
   ```
   发现以下问题：
   - 重复记忆 3 条（用户喜欢跑步 x3）→ 建议合并为1条
   - 无标签记忆 5 条 → 建议补充分类
   - 冲突记忆 1 条 → 需要您确认保留哪条
   ```

4. **用户确认后执行**
   - 删除重复项：调用 `mcp__qdrant-memory__delete`（保留最新/质量最好的一条）
   - 重新存储优化版本：按操作一的格式重新存入

5. **汇报结果**：
   ```
   整理完成 ✅
   - 删除重复记忆: 5 条
   - 优化格式: 3 条
   - 合并冲突: 1 条
   ```

---

## 操作三：更新/删除记忆

当用户说"我不再喜欢 XX 了"或"更新 XX 的记忆"时：

1. 搜索相关记忆，找到最匹配的条目
2. 展示给用户确认
3. 删除旧记忆（`mcp__qdrant-memory__delete`）
4. 如果是更新（不是删除），按操作一的格式存入新版本

---

## 同义词扩展指南

扩展时遵循"**紧密相关**"原则，3~5个词，不过度联想：

| 原始信息 | ✅ 合适的扩展 | ❌ 过度扩展 |
|---------|------------|-----------|
| 喜欢走路 | 步行、散步、徒步、有氧运动 | 健康、减肥、大自然 |
| 喜欢苹果 | 苹果、果类、水果、青苹果 | 健康饮食、维生素 |
| 喜欢玩游戏 | 游戏、电子游戏、电玩、手游、PC游戏 | 娱乐、休闲、宅 |
| 叫志远 | 姓名、名字、称呼 | 身份、个人 |

扩展词要能真实提升搜索命中，而不是泛化到毫无区分度的词。
